# Тренировки — планировщик недели

Личный сайт для планирования силовых и кардио-тренировок на неделю с
визуальным отслеживанием проработки групп мышц.

Живая версия: https://volkovidpm.github.io/personal_training/

## Возможности

- **Сетка недели** (Пн–Вс) с переключением между неделями и историей.
- **Библиотека упражнений** (~34 стартовых) с техникой выполнения, группами
  мышц и нужным оборудованием. Можно добавлять свои упражнения.
- **Панель проработки**: анатомическая карта тела (вид спереди и сзади) с
  тепловой подсветкой + полоски интенсивности по каждой группе мышц.
- **Расчёт нагрузки по объёму**: подходы × повторы (силовые) или минуты
  (кардио). Основные мышцы — вес ×2, вспомогательные — ×1.
- **Экспорт/импорт** всех данных в JSON для бэкапа между устройствами.

Все данные хранятся локально в браузере (localStorage), без сервера.

Идеи и цели на будущее — в [ROADMAP.md](ROADMAP.md).

## Стек

React + TypeScript + Vite + Zustand. Тесты — Vitest.

## Разработка

```bash
npm install
npm run dev      # дев-сервер
npm test         # юнит-тесты
npm run build    # типизация + продакшен-сборка
```

## Деплой

Автоматический через GitHub Actions (`.github/workflows/deploy.yml`) при пуше в
`main`: тесты → сборка → публикация `dist/` на GitHub Pages. Базовый путь
`/personal_training/` задаётся в `vite.config.ts`.

## Структура

- `src/data/exercises.ts` — стартовая библиотека упражнений.
- `src/lib/` — доменная логика: расчёт нагрузки, группы мышц, ISO-недели.
- `src/state/` — хранилища Zustand и обёртка над localStorage.
- `src/components/`, `src/views/` — UI (сетка недели, карта тела, библиотека).
