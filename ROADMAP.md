# Роадмап

Идеи и цели на будущее для планировщика тренировок. Список живой — дополняй и
вычёркивай по мере работы. Приоритет грубый: сверху — что полезнее в первую
очередь.

## Контент: библиотека упражнений

- [ ] **Расширить библиотеку** — добавить упражнения сверх стартовых 34:
  вариации (узкий/широкий хват, в наклоне/сидя), упражнения с резинками,
  с собственным весом, на TRX, гири.
- [ ] **Улучшить описания техники** — сделать пошаговыми (нумерованные шаги:
  исходное положение → движение → дыхание → типичные ошибки).
- [ ] **Картинки/анимации к упражнениям** — хотя бы иконка или схема движения;
  можно ссылками на видео (YouTube) в поле упражнения.
- [ ] **Уровень сложности** упражнения (новичок / средний / продвинутый) и
  фильтр по нему в библиотеке.
- [ ] **Теги/категории** помимо групп мышц: «база», «изоляция», «дома», «зал».

## Планирование тренировок

- [ ] **Шаблоны тренировок (сплиты)** — готовые программы (full body, верх/низ,
  push/pull/legs); применить шаблон на неделю одной кнопкой.
- [ ] **Копирование недели** — продублировать прошлую неделю в текущую/следующую.
- [ ] **Копирование/перетаскивание упражнения** между днями (drag & drop).
- [ ] **Заметки к дню** (самочувствие, вес снаряда по факту).
- [ ] **Рабочий вес** в записи (кг), а не только подходы×повторы.

## Аналитика и прогресс

- [ ] **График прогресса по неделям** — динамика нагрузки на группу мышц,
  суммарный объём, минуты кардио.
- [ ] **Цели по неделе** — задать целевую нагрузку на группу, показывать % выполнения.
- [ ] **Умные подсказки баланса** — не просто «не задействованы», а «спина
  недогружена относительно груди» (баланс антагонистов).
- [ ] **Настраиваемые пороги интенсивности** (`lib/load.ts` уже параметризован —
  вынести в настройки/UI).
